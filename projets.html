Développement de mon application mobile E-sport

Etant fan d'e-sport j'ai décidé de me lancer dans le développement d'une application mobile en Novembre 2021. Au départ l'idée était de la développer avec le langage Dart et le framework Flutter cependant ne disposant pas d'un ordinateur Mac j'ai décidé de la développer sur Android. L'application est toujours en cours de développement et je souhaite mettre en ligne sa 1ère version pour Décembre 2022.

Durant le développement de mon application j'ai dû apprendre à utiliser l'éditeur de code Android Studio, c'est un environnement avec lequel je deviens de plus en plus familier et que je trouve très complet notamment pour communiquer avec Firebase. J'ai développer l'application en Kotlin, c'est un langage officiellement adopté par Android après Java. Avec Kotlin j'ai appris créer des boutons, des menus, gérer une interface, gérer l'authentification d'un utilisateur avec Firebase, utiliser les différents composants d'une application (App bar, Bottom navigation, Navigation drawer, Tabs/TabLayout) dans les Activities et les Fragments.

Apprentissage des bases de la programmation

Ayant débuté mon parcours dans le développement avec Kotlin j'ai rencontré beaucoup de difficultés pour comprendre ce que je faisais. Pour acquérir les bases j'ai décidé d'apprendre le langage de balisage HTML et le CSS. Nous avions des cours de programmation durant ma formation à l'école cependant j'ai décidé d'aller plus loin en suivant une formation en ligne. D'une part grâce à mes cours j'ai appris à me familiariser avec Visual Studio Code en comprenant son fonctionnement et ses raccourcis. Ensuite nous avons appris les bases, structurer son site web (header, main, footer), créer des classes/id, créer des tableaux, intégrer des images et/ou vidéos, intégrer du CSS/Javascript à son code.

Ensuite concernant le CSS je suis allé plus loin que mes cours en suivant une formation. Le contenu de la formation était composé de : les sélecteurs simple en CSS, les sélecteurs d'attributs/id/class/, les pseudo-classes, les couleurs, les propriétés de typographie/bordures/marges, le flux et positionnement flottant, les unités de mesures, du Flexbox, de CSS Grid, du flux et positionnement flottant et du Responsive.

Le développement Web avec JAVASCRIPT

Le premier langage que j'ai abordé en développement web est Javascript. Pour cela, j'ai suivi une formation en ligne payante sur la plateforme Udemy. Cette formation dure 135h et aborde le Javascript, Bootstrap, PHP, une introduction à Node.js/Angular/MongoDB/, MySQL. J'ai réalisé plusieurs projets dont 2 qui mon réellement permis de comprendre Javascript.

Mon premier projet en Javascript est une To do list. Je considère que s'exercer à réaliser une to do list est un bon exercice car j'ai découverts comment utiliser les sélecteurs, créer des évènements (addEventListener) ou encore créer des fonctions avec des conditions.

out d'abord en réalisant ce projet j'ai poussé mes connaissances en Javascript, j'ai appris ce qu'était le DOM, utilisé des constantes et des variables et compris leurs différences, durant le développement de l’application j'ai fait appel à une API ce qui m'a permis de comprendre leur principe fonctionnement.
